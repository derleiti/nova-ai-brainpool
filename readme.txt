Nova AI Brainpool ist ein minimalistisches AI-Chat Plugin im Retro-Terminal-Stil – powered by AILinux & Ollama.

== Description ==

🌐 *Nova AI Brainpool* bringt dir einen simplen, aber mächtigen Chatbot in deinen WordPress-Auftritt – perfekt für Support, Beratung oder Interaktion.  
🏗️ Entwickelt für AILinux & Ollama, läuft komplett lokal – kein Cloud-Zwang.  
🎨 Design im klassischen Terminal-Style: schnell, schlicht, nerdig geil.

**Features:**
- Terminal-inspirierter Stil (Grün auf Schwarz)
- AI-Chat mit Ollama Backend (z. B. Mistral, LLaMA, etc.)
- Keine externen APIs – du behältst die Kontrolle
- Plugin-Settings für API-URL
- Kurzbefehl: `[nova_ai_chat]`
- Leicht erweiterbar mit eigenem Theme

== Installation ==

1. Plugin hochladen oder aus ZIP installieren  
2. Aktivieren  
3. Im Menü "Nova AI" die API-URL prüfen/anpassen (Default: `http://host.docker.internal:11434/api/generate`)  
4. Den Shortcode `[nova_ai_chat]` auf einer Seite oder in einem Beitrag verwenden  
5. Ollama starten und Modell bereitstellen (z. B. `mistral`)  
6. Fertig 🤖
